**主な機能：**
- ✅ 完全なチェスルール実装
- ✅ すべての駒の動き（キング、クイーン、ルーク、ビショップ、ナイト、ポーン）
- ✅ チェックとチェックメイトの検出

**遊び方：**
1. 駒をクリックして選択
2. 緑色にハイライトされたマスが移動可能な場所
3. 赤い枠は相手の駒を取れる場所
4. 目標：相手のキングをチェックメイトにする
